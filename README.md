# 📚 Library Management System (LMSWebApp)

A Library Management System developed using **C#** for backend logic and **HTML & CSS** for the user interface.  
The application efficiently manages library operations such as book records, user handling, and basic data organization.

---

## 🚀 Features

- Add, update, and manage library records
- User-friendly interface built with HTML & CSS
- Structured backend using C#
- Separation of concerns with Data Access Layer
- Efficient and reliable system performance
- Clean and maintainable project structure

---

## 🛠 Tech Stack

- **Backend:** C#
- **Frontend/UI:** HTML, CSS
- **Architecture:** Layered Architecture (Data Access Layer)
- **Tools:** Visual Studio, .NET
